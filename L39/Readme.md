
# Library Project

## Описание
Проект для управления библиотекой с использованием Java, Maven и Docker.

### Запуск тестов
```bash
mvn test
```
*Запускает unit-тесты приложения*

### Сборка приложения
```bash
mvn package
```

### Запуск в Docker
```bash
docker-compose up -d
```
*Запускает контейнеры в фоновом режиме*

### Остановка приложения
```bash
docker-compose down -v
```
*Останавливает контейнеры и удаляет тома*

## Доступ к приложению
После запуска приложение доступно по адресу:  
http://localhost:8080/library/api/*

где * = books / readers / borrowed

## Тестирование postman

---

**1. Добавить книгу (POST)**

*   **Метод:** `POST`
*   **URL:** `http://localhost:8080/library/api/books`
*   **Body (raw JSON):**
    ```json
    {
      "title": "Война и мир",
      "author": "Лев Толстой",
      "published_year": 1869,
      "genre": "Роман"
    }
    ```
*   **Ожидаемый результат:** Статус `201 Created`. В теле ответа должен быть JSON созданной книги с присвоенным `book_id`.

**2. Получить все книги (GET)**

*   **Метод:** `GET`
*   **URL:** `http://localhost:8080/library/api/books`
*   **Ожидаемый результат:** Статус `200 OK`. В теле ответа должен быть JSON-массив всех книг.

**3. Получить книгу по ID (GET)**

*   **Метод:** `GET`
*   **URL:** `http://localhost:8080/library/api/books/{id}`
    *   Замените `{id}` на **фактический ID** книги (например, `1`).
*   **Ожидаемый результат:** Статус `200 OK`. В теле ответа должен быть JSON книги с указанным ID. Если книга не найдена, статус `404 Not Found`.

**4. Добавить читателя (POST)**

*   **Метод:** `POST`
*   **URL:** `http://localhost:8080/library/api/readers`
*   **Body (raw JSON):**
    ```json
    {
      "name": "Иванов Иван",
      "email": "ivanov@example.com",
      "phone": "+79123456789"
    }
    ```
*   **Ожидаемый результат:** Статус `201 Created`. В теле ответа должен быть JSON созданного читателя с присвоенным `reader_id`.

**5. Получить всех читателей (GET)**

*   **Метод:** `GET`
*   **URL:** `http://localhost:8080/library/api/readers`
*   **Ожидаемый результат:** Статус `200 OK`. В теле ответа должен быть JSON-массив всех читателей.

**6. Получить читателя по ID (GET)**

*   **Метод:** `GET`
*   **URL:** `http://localhost:8080/library/api/readers/{id}`
    *   Замените `{id}` на **фактический ID** читателя (например, `1`).
*   **Ожидаемый результат:** Статус `200 OK`. В теле ответа должен быть JSON читателя с указанным ID. Если читатель не найден, статус `404 Not Found`.

**7. Выдать книгу (POST)**

*   **Метод:** `POST`
*   **URL:** `http://localhost:8080/library/api/borrowed`
*   **Body (raw JSON):**
    *   Замените `bookId` и `readerId` на **фактические ID**, полученные при добавлении книги и читателя.
    ```json
    {
      "bookId": 1,
      "readerId": 1
    }
    ```
*   **Ожидаемый результат:** Статус `201 Created`. В теле ответа должен быть JSON записи о выдаче с присвоенным `borrow_id`, статусом `borrowed` и текущей датой `borrow_date`.

**8. Получить все выданные книги (GET)**

*   **Метод:** `GET`
*   **URL:** `http://localhost:8080/library/api/borrowed`
*   **Ожидаемый результат:** Статус `200 OK`. В теле ответа должен быть JSON-массив всех записей о выдаче/возврате книг.

**9. Вернуть книгу (PUT)**

*   **Метод:** `PUT`
*   **URL:** `http://localhost:8080/library/api/borrowed/{id}/return`
    *   Замените `{id}` на **фактический ID** записи о выдаче (например, `1`).
*   **Ожидаемый результат:** Статус `200 OK`. В теле ответа должен быть JSON с сообщением об успешном возврате. Статус книги в базе данных должен измениться на `returned`. Если запись не найдена или возникла ошибка, возможен статус `400 Bad Request` или `404 Not Found`.

**10. Получить книги, находящиеся у читателя (GET)**

*   **Метод:** `GET`
*   **URL:** `http://localhost:8080/library/api/books/{readerId}/borrowed`
    *   Замените `{readerId}` на **фактический ID** читателя, которому была выдана книга (например, `1`).
*   **Ожидаемый результат:** Статус `200 OK`. В теле ответа должен быть JSON-массив с книгами, которые находятся у указанного читателя (имеют статус `borrowed`). Если у читателя нет книг на руках, будет возвращен пустой массив `[]`.
